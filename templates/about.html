ℹ️ About This Project

This Advanced House Price Predictor is built as a data science project. It uses a Linear Regression Machine Learning model trained on geographically calibrated data points across different country, state, and district tiers.

By implementing custom constraint logic, the AI scales housing valuations logically—ensuring rural tiers (like Nirmal) remain bounded realistically while commercial metro hubs scale dynamically according to real estate market metrics.
